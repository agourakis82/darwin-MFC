## Gap Analysis

To reach the 85% F1 target, we need:

1. **Reduce False Positives (currently 290)**
   - Regex patterns are too broad
   - Need contextual understanding to avoid over-matching

2. **Reduce False Negatives (currently 231)**
   - Many entities not covered by regex patterns
   - Variants, abbreviations, and synonyms not handled

3. **Improve EXAM Recognition**
   - Current F1: 19.4%
   - Many exam formats not captured (e.g., "PA", "FC", specific lab values)

4. **Improve DISEASE Recognition**
   - Current recall: 24.3%
   - Many disease names not in pattern library
