## Implementation Notes

### Google Forms Setup

1. Create form in primary language (Portuguese)
2. Enable "Collect email addresses" OPTIONAL
3. Set "Limit to 1 response" OFF (anonymous)
4. Create language-specific copies
5. Test on low-bandwidth connections

### Distribution Channels

**Primary:**
- WhatsApp groups (medical associations, alumni networks)
- Partner organization mailing lists

**Secondary:**
- Twitter/X announcements
- LinkedIn professional groups
- Medical school networks

### QR Code Strategy

Generate QR codes for each language version to facilitate:
- Conference distribution
- Print materials
- Partner organization sharing

## Translation Guidelines

1. Maintain meaning over literal translation
2. Adapt cultural references appropriately
3. Use local medical terminology where applicable
4. Test with native speaker practitioners
5. Keep response options parallel across languages

## Data Collection

All responses funnel to single analysis dataset with:
- Language identifier
- Timestamp
- Anonymized responses

## Privacy Compliance

- No PII required (email optional)
- GDPR-compliant data handling
- Explicit consent in survey intro
- Right to withdrawal explained
